## Cómo tomar las capturas

1. Abre KronoSync en tu equipo.
2. Navega a cada pantalla listada arriba.
3. Usa `Win + Shift + S` para recortar la sección relevante.
4. Guarda la imagen en formato PNG en esta carpeta (`docs/img/`).
5. Usa nombres exactos (sin espacios, distingue mayúsculas/minúsculas).

## Dimensiones recomendadas

- Ancho: 800-1200px
- Formato: PNG
- Las imágenes de la interfaz deben capturar solo la ventana de KronoSync (sin el escritorio de fondo si es posible).
